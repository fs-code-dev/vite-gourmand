# README – Vite & Gourmand

## Présentation du projet

**Vite & Gourmand** est une entreprise familiale située à Bordeaux depuis 25 ans. Le site web a été développé dans le cadre du titre professionnel **Développeur Web et Web Mobile**.

L’objectif de cette application est de permettre aux visiteurs de consulter les menus proposés par l’entreprise et de commander facilement en ligne.

Le site propose :

* une présentation de l’entreprise
* une consultation des menus
* une vue détaillée des prestations
* un système de commande
* un espace utilisateur
* un espace employé et administrateur
